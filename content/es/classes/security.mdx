# Security — Clase Tier B

La clase Security te otorga un X-Taser desde el principio para neutralizar amenazas y eventos disruptivos. Sin embargo, el taser también está disponible de forma gratuita en la Oficina de Recepción, lo que hace que el precio de 1.250 Animal Coins sea difícil de justificar para la mayoría de los jugadores.

## Resumen

| Atributo | Valor |
|-----------|-------|
| **Costo** | 1.250 Animal Coins |
| **Tier** | B |
| **Rol** | Defensa / Control de Amenazas |
| **Rareza** | Estándar |
| Cordura | Baja (Nivel 1), Media (Nivel 2–3) |
| Utilidad | Media-Alta |

## Habilidad Principal

Comienza cada ronda con un X-Taser (5 usos) — un arma paralizante que puede incapacitar amenazas y entidades disruptivas en el hospital.

## Niveles de Mejora

### Nivel 1
Comienza con X-Taser (5 usos)

### Nivel 2
Comienza con X-Taser (8 usos), +10 de Cordura Bonus

### Nivel 3
Comienza con X-Taser (8 usos), +15 de Cordura Bonus

## Consejos de Estrategia

El mayor argumento en contra de Security es que el X-Taser aparece gratis en la Oficina de Recepción cada ronda. Cualquier clase puede tomarlo, lo que significa que estás pagando 1.250 monedas por la conveniencia de tenerlo en tus manos desde el principio. Si tu equipo se coordina, un jugador puede simplemente recoger el taser gratis cada ronda sin dedicar un espacio de clase a Security.

Dicho esto, Security se vuelve más atractivo en partidas caóticas donde el taser gratis es arrebatado por otro jugador o destruido. La mejora de Nivel 2, que añade 3 usos extra y +10 de Cordura Bonus, hace que la clase sea significativamente más autosuficiente, y los +15 de Cordura del Nivel 3 significan que puedes patrullar por más tiempo. Si juegas frecuentemente en salas llenas con jugadores no coordinados, el acceso garantizado al taser es una verdadera mejora en la calidad de vida.

## Comparación con Clases de Tier Adyacente

| Clase | Costo | Beneficio Principal |
|-------|------|-------------|
| **Doctor** | 900 Monedas | Cordura al Curar |
| **Security** | 1.250 Monedas | X-Taser al Aparecer |
| **Surgeon** | ~2.000+ Monedas | Curación Avanzada |

Con 1.250 monedas, Security cuesta más que Doctor pero ofrece un valor menos consistente. La recuperación de Cordura del Doctor es pasiva y siempre está activa; el taser del Security es situacional y redundante si alguien toma el gratis. El Surgeon, aunque más caro, proporciona una utilidad de curación drásticamente mejor. Security es una opción de nicho — fuerte para jugadores que aman patrullar y manejar amenazas, pero no se recomienda como clase de propósito general.
