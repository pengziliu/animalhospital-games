# Security — B Tier Class

The Security class gives you an X-Taser right out of the gate for neutralizing threats and disruptive events. However, the taser is also available for free in the Reception Office, which makes the 1250 Animal Coin price tag hard to justify for most players.

## Overview

| Attribute | Value |
|-----------|-------|
| **Cost** | 1,250 Animal Coins |
| **Tier** | B |
| **Role** | Defense / Threat Control |
| **Rarity** | Standard |
| Sanity | Low (Level 1), Medium (Level 2–3) |
| Utility | Medium-High |

## Core Ability

Start each round with an X-Taser (5 uses) — a stun weapon that can incapacitate threats and disruptive entities in the hospital.

## Upgrade Levels

### Level 1
Start with X-Taser (5 uses)

### Level 2
Start with X-Taser (8 uses), +10 Bonus Sanity

### Level 3
Start with X-Taser (8 uses), +15 Bonus Sanity

## Strategy Tips

The biggest argument against Security is that the X-Taser spawns for free in the Reception Office every round. Any class can grab it, meaning you are paying 1,250 coins for the convenience of having it in your hands from the start. If your team coordinates, one player can simply pick up the free taser each round without dedicating a class slot to Security.

That said, Security becomes more appealing in chaotic games where the free taser gets snatched by another player or destroyed. The Level 2 upgrade adding 3 extra uses and +10 Bonus Sanity makes the class significantly more self-sufficient, and Level 3's +15 Sanity means you can stay on patrol longer. If you frequently play in full lobbies with uncoordinated players, the guaranteed taser access is a real quality-of-life improvement.

## Comparison to Adjacent Tier Classes

| Class | Cost | Key Benefit |
|-------|------|-------------|
| **Doctor** | 900 Coins | Sanity on Heal |
| **Security** | 1,250 Coins | X-Taser on Spawn |
| **Surgeon** | ~2,000+ Coins | Advanced Healing |

At 1,250 coins, Security costs more than Doctor but offers less consistent value. The Doctor's Sanity recovery is passive and always active; the Security's taser is situational and redundant if someone grabs the free one. The Surgeon, while more expensive, provides dramatically better healing utility. Security is a niche pick — strong for players who love patrolling and handling threats, but not recommended as a general-purpose class.
