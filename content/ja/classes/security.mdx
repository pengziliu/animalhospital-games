# セキュリティ — Bティアクラス

セキュリティクラスは、脅威や破壊的なイベントを無力化するためのXテーザーをゲーム開始直から提供します。ただし、テーザーは受付オフィスでも無料で入手できるため、1250アニマルコインという価格はほとんどのプレイヤーにとって正当化するのが難しいでしょう。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 1,250 アニマルコイン |
| **ティア** | B |
| **役割** | 防御 / 脅威コントロール |
| **レアリティ** | スタンダード |
| 正気度 | 低（レベル1）、中（レベル2〜3） |
| 効用 | 中〜高 |

## コアアビリティ

毎ラウンドXテーザー（5回使用可能）で開始 — 病院内の脅威や破壊的なエンティティを無力化できるスタン武器。

## アップグレードレベル

### レベル1
Xテーザー（5回使用可能）で開始

### レベル2
Xテーザー（8回使用可能）で開始、+10ボーナス正気度

### レベル3
Xテーザー（8回使用可能）で開始、+15ボーナス正気度

## 戦略のヒント

セキュリティに対する最大の反論は、Xテーザーが毎ラウンド受付オフィスに無料で出現することです。どのクラスでも拾うことができるため、1250コインを支払うのは最初から手元にあるという利便性のためだけということになります。チームが連携できていれば、セキュリティにクラス枠を割くことなく、毎ラウンド1人のプレイヤーが無料のテーザーを拾うだけで済みます。

とはいえ、無料のテーザーが他のプレイヤーに奪われたり破壊されたりする混沌としたゲームでは、セキュリティはより魅力的になります。レベル2のアップグレードで3回分の追加使用回数と+10のボーナス正気度が得られ、クラスの自立性が大幅に向上します。さらにレベル3の+15正気度により、より長く巡回を続けることができます。連携の取れていないプレイヤーがいる満員のロビーで頻繁にプレイする場合、テーザーへの確実なアクセスは生活の質（QoL）の向上として実感できるでしょう。

## 隣接ティアクラスとの比較

| クラス | コスト | 主なメリット |
|-------|------|-------------|
| **ドクター** | 900コイン | 回復時の正気度 |
| **セキュリティ** | 1,250コイン | 出現時のXテーザー |
| **サージョン** | 約2,000+コイン | 高度な回復 |

1,250コインのセキュリティは、ドクターよりもコストが高いですが、一貫した価値は低くなります。ドクターの正気度回復はパッシブで常に発動していますが、セキュリティのテーザーは状況に左右され、誰かが無料のものを拾った場合は冗長になります。サージョンはより高額ですが、劇的に優れた回復効用を提供します。セキュリティはニッチな選択であり、巡回や脅威への対処を好むプレイヤーには強力ですが、汎用クラスとしては推奨されません。
