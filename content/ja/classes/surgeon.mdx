# サージャン — Sティアクラス

アニマルホスピタルにおける非Robuxプレイヤー向けのトップクラス。すべての回復でパッシブにSAN値を獲得しスピードブーストを得るため、より多くの患者を治療するにつれて自己完結できるようになります。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 2500 アニマルコイン |
| **ティア** | S |
| **役割** | ヒーラー / メイン患者治療 |
| **レアリティ** | プレミアム（最高コスト） |
| SAN値 | 中〜高 |
| 利便性 | 中〜低 |
| 開始資金 | なし |


## コアアビリティ

患者を治療した後、SAN値と短時間のスピードブーストを獲得

## アップグレードレベル

### レベル1
患者を治療した後、SAN値と短時間のスピードブーストを獲得

### レベル2
患者を治療した後、SAN値とより長時間（8秒）のスピードブーストを獲得

### レベル3
患者を治療した後、SAN値x2とより長時間（8秒）のスピードブーストを獲得、さらに+10のボーナスSAN値

## 戦略のヒント

ゲーム内で最も優れた無料プレイクラスです。患者の治療を優先し、SAN値+スピードブーストを常に発動させましょう.
