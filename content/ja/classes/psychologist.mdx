# サイコロジスト — Aティアクラス

ハイリスク・ハイリターンの選択。SAN値の効果が2倍になるため、ポジティブな要素からはより早く回復しますが、ネガティブな要素からはより多くのダメージを受けます。レベル2以上では、SAN値の減少を完全に無視できる確率が追加されます。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 500 アニマルコイン |
| **ティア** | A |
| **ロール** | サポート / リスク＆リターン型SAN値管理 |
| **レアリティ** | スタンダード |
| SAN値 | 変動（ハイリスク/ハイリターン） |
| 利便性 | 中 |
| 所持金 | なし |


## コアアビリティ

すべてのSAN値の効果が2倍になる（ポジティブおよびネガティブ両方）

## アップグレードレベル

### レベル1
すべてのSAN値の効果が2倍になる（ポジティブおよびネガティブ両方）

### レベル2
すべてのSAN値の効果が2倍になる（ポジティブおよびネガティブ両方）、7.5%の確率でSAN値の減少を無視

### レベル3
すべてのSAN値の効果が2倍になる（ポジティブおよびネガティブ両方）、15%の確率でSAN値の減少を無視

## 戦略のヒント







ハイリスク・ハイリターン。コーヒーやチョコレートなどのSAN値回復アイテムと非常に相性が良いです。2倍の効果により、ほぼ無敵になれるかもしれません — しかし、油断すれば破滅することもあります。
