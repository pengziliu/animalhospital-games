# Animal Hospital FAQ — よくある質問

RobloxのAnimal Hospitalについて知るべきすべてを回答します。初心者向けの基本的な質問から、ベテラン向けの高度なメカニクスまで網羅しています。

---

## 一般的な質問

### Animal Hospitalとは？

**Animal Hospital**（別名：Animal Hospital: Anomaly）は、動物病院で夜勤を務める**協力型サバイバルホラーゲーム**です。あなたの仕事は、動物の患者の中に隠れたアノマリー（普通の動物に変装したモンスター）を見つけ出し、Sanity（正気度）を維持しながらシフトを生き延びることです。

- **開発元**: Animal Anomaly（Roytt主導）
- **リリース日**: 2026年5月10日
- **ジャンル**: 協力型サバイバルホラー
- **総来場者数**: 45.2M以上
- **最高同時接続数（CCU）**: 135,000人以上
- **評価**: 95.8%が高評価

### Animal Hospitalは無料で遊べますか？

**はい、100%無料です。** 課金壁やプレミアムな参加条件はありません。Robloxアカウントを持っていれば誰でもすぐに遊ぶことができます。オプションでの購入（Robuxでのクラスやアイテムの購入）はゲームプレイを強化しますが、フル体験を楽しむために必須ではありません。

### 何人まで一緒に遊べますか？

1サーバーあたり最大**30人**のプレイヤーが遊べます。ソロプレイとフル協力プレイの両方をサポートしています。3〜6人の少人数グループは、連携が取りやすいため最も良いパフォーマンスを発揮する傾向があります。

### Animal Hospitalはモバイルで遊べますか？

**はい。** Robloxアプリを通じてモバイル端末で遊ぶことができます。ただし、アノマリー検出に必要な正確なカメラ操作と素早い反応時間が必要なため、PCでのプレイが最も推奨されます。モバイルプレイヤーもすべてのゲームモードに完全に参加できます。

### Animal Hospitalの年齢制限はどうなっていますか？

Animal Hospitalにはジャンプスケア、超自然的なテーマ、モンスターとの遭遇などのホラー要素が含まれています。ほとんどのRobloxプレイヤーに適していますが、小さな子供には怖すぎる場合があります。

## ゲームプレイの質問

### アノマリーとは何ですか？

**アノマリー**は、通常の動物の患者に変装したモンスターです。彼らは通常の患者に紛れ込み、3つの検出方法でのみ識別できます：

1. **外見** — 受付で患者を直接観察する（例：3つの目、うつろな顔、痙攣）
2. **写真** — 机に現れるポラロイド写真を確認する（例：静電写真、呪われた写真、目の違い）
3. **CCTV** — 監視カメラで確認する（例：虚無の体、スキンウォーカー、カメラを見つめる）

アノマリーを通してしまうと、チームのSanityが減少し、危険なイベントが発生する可能性があります。

### Sanity（正気度）はどのように機能しますか？

**Sanityはあなたのヘルスバーです。** 0になると、シフトから排除されます。

| Sanityを減少させる要因 | 減少量 |
|--------------------|--------|
| 呪われた写真（拾った時） | -10 |
| カメラアノマリー / デーモンキャット（ジャンプスケア） | -20 |
| ザ・ストーカー（目を合わせる） | 視線につき -10 |
| 未確認のアノマリー | 異なる |
| Don't Look Up（天井を見る） | 継続的に減少 |
| 通常の露出（時間経過） | ゆっくり減少 |

| Sanityを回復させる要因 | 回復量 |
|---------------------|--------|
| コーヒー（1口あたり） | +5（3口 = 合計15） |
| チョコレート（1口あたり） | +20（3口 = 合計60） |
| ドクター / サージョンの回復 | レベルによって異なる |
| 秘書のチェックイン | レベルによって異なる |
| サイコログストクラス | すべてのSanity効果が2倍 |

### Sanityが0になるとどうなりますか？

Sanityが**0**に達すると、**現在のシフトから排除**されます。次のシフトが始まるまで参加できなくなります。これは事実上の「死」であり、チームメンバーを失うことで他の全員のアノマリー確認がより困難になります。

### スキンウォーカーにはどう対処すればよいですか？

**スキンウォーカー**は、ゲーム内で最も危険なモンスターの1つです。以下の方法で対処してください：

| 方法 | 効果 |
|--------|-------------|
| **銃（Gun）** | 撃つ — 20発撃てる、最も確実な方法 |
| **スタンガン（Taser）** | X-Taserで気絶させる（5回使用可能、受付オフィスで無料） |
| **Eキーを長押し** | スキンウォーカーを犠牲者から強引に引き剥がす |
| **Eキー連打** | 直接攻撃された場合、スキンウォーカーをパンチする |
| **タイマー内に倒す** | タイマーが切れる前に遭遇を終わらせる — さもなくばあなたまたは患者を食べられる |

シークレットエージェントクラスは、内蔵の銃があるため、スキンウォーカーに対する最も優れたカウンターとなります。

### ザ・ストーカーとは何ですか？

**ザ・ストーカー**は、病院内のあちこちからあなたを見つめるモンスターです。**目を合わせる**と、見つめられるごとに**10 Sanity**を失います。最善の戦略は単に**目を逸らす**ことです。致命的ではありませんが、シフト中にSanityの減少が積み重なっていきます。

## クラスと進行

### 最強のクラスは何ですか？

| カテゴリ | 最強クラス | 理由 |
|----------|-----------|-----|
| **総合最強（無料）** | **サージョン（Surgeon）** (2,500 AC) | 回復のたびにSanityが回復し、スピードブーストが付与される。レベル3でSanity獲得量が2倍になる。 |
| **総合最強（有料）** | **シークレットエージェント（Secret Agent）** (320–890 R$) | 20発撃てる銃が内蔵。レベル3で弾が再生する。 |
| **最高のコストパフォーマンス** | **ドクター（Doctor）** (900 AC) | 回復時にSanityが回復する。手頃で効果的。 |
| **最高のサポート** | **サイコログスト（Psychologist）** (500 AC) | Sanity効果が2倍 — コーヒーやチョコレートの価値が2倍になる。 |
| **最高の初期クラス** | **インターン（Intern）** (無料) | ボーナスSanity +10。コスト不要で、初心者にとって良い受け皿となる。 |

### クラスはどのようにアンロックしますか？

クラスは**ロビーのクラショップ**で以下を使用して購入します：

- **Animal Coins**（シフト完了によってゲーム内で獲得）
- **Robux**（プレミアムRoblox通貨、現実のお金）

各クラスには**3つのアップグレードレベル**があり、徐々により強力な効果が得られます。

### おすすめのクラス進行順は？

```
インターン（無料） → ドクター（900 AC） → サージョン（2,500 AC） → [オプション] シークレットエージェント（Robux）
```

このルートでは、基本的な無料クラスから、獲得可能な最強のクラスへと進み、その後オプションで最強の有料戦闘クラスを追加します。

## 経済と通貨

### Animal Coinsはどのように獲得できますか？

Animal Coinsは**シフトを完了すること**で獲得できます。金額は以下の要素によって異なります：

- **使用したクラス** — 上位クラスほど獲得量が多くなる傾向
- **シフトの長さ** — 長いシフト = 多くのコイン
- **効率** — より多くの患者を治療する = より高い報酬
- **アノマリーの捕捉** — アノマリーを正しく特定するとスコアに貢献

ほとんどのプレイヤーは、クラスとパフォーマンスに応じて1シフトあたり**50〜200 Animal Coins**を獲得します。

詳細な獲得戦略については、[Animal Coinsガイド](/guide/animal-coins-guide)をご覧ください。

### Animal Hospitalにコードはありますか？

**いいえ。** 2026年6月現在、Animal Hospitalには**コード引換システムがありません**。有効なプロモコードはなく、ゲーム内にコード入力機能も存在しません。

今後のコードに関する情報を入手するには：
- **Animal HospitalのDiscord**サーバーに参加する
- **Animal AnomalyのRobloxグループ**をフォローする
- Robloxページのゲーム説明を確認する

コードがリリースされた瞬間に[コードページ](/codes/codes)を更新します。

### Animal Coinsは最初に何に使うべきですか？

| 優先順位 | 購入対象 | コスト |
|----------|----------|------|
| 1番目 | **ドクタークラス** | 900 AC |
| 2番目 | **サージョンクラス** | 2,500 AC |
| 3番目 | **サイコログストクラス**（オプション） | 500 AC |

早い段階では積極的に貯金しましょう。コインが貯まるまでの最初の数回のシフトは、インターンで遊んでも全く問題ありません。

## 高度な質問

### ソロで遊べますか？

**はい。** Animal Hospitalは完全にソロプレイ可能です。最もおすすめのソロクラスは、自己完結型のSanity回復ができる**サージョン**です。ソロプレイの主な課題は以下の通りです：

- すべての確認を一人で行う（並行してCCTVを見る人がいない）
- モンスターとの遭遇（スキンウォーカー、ハイダー）がチームのサポートなしではより危険
- プレイヤーが一人しかいないため、患者の処理スピードが遅くなる

### Animal Coinsとドル（ゲーム内現金）の違いは何ですか？

| 通貨 | 獲得方法 | 用途 |
|----------|----------|----------|
| **Animal Coins (AC)** | シフトの完了 | 永続的なクラスの購入 |
| **ゲーム内ドル ($)** | 患者の治療、シフト報酬 | アップグレードや消耗品の購入 |

Animal Coinsは**プレミアム進行通貨**です。クラスのためにとっておきましょう。ドルは**シフトごとの通貨**であり、アップグレードやアイテムに自由に使えます。

### アノマリー検出を上達させるにはどうすればよいですか？

1. **毎回3ステップのプロトコルに従う** — 外見 → 写真 → CCTV
2. **外見チェックに4〜5秒かける** — 痙攣テストを急がない
3. **拾う前に必ず机の上の写真を検査する** — 呪われた写真のダメージを防ぐ
4. **すべての患者の待合室CCTVを確認する** — これが最も重要なカメラです
5. **アノマリーの種類を暗記する** — [アノマリーチートシート](/guide/how-to-identify-anomalies)を活用

### モンスターや危険要素にはどんなものがありますか？

| モンスター | 危険度 | 対処法 |
|---------|-------------|---------------|
| **ベッドモンスター** | 高 | ベッドを飛び越える、赤い線を避ける、またはメープルシロップを使う |
| **スキンウォーカー** | 非常に高い | 銃、スタンガン、Eキー長押しで引き剥がす |
| **ハイダー / 半透明の男** | 高 | ロビーへ逃げる、銃/スタンガンを使う、または避ける |
| **ザ・ストーカー** | 中 | 目を合わせない |
| **Don't Look Up** | 中 | 目薬を使う、天井を見ない |
| **カメラアノマリー / デーモンキャット** | 中 | すばやくExitボタンを押す（逃すと-20 Sanity） |
| **テンドリル** | 中 | タイマーが切れる前に手術を完了させる |
| **ヘッドバンガー** | 低 | 後ろに反り返った瞬間にシャッターを閉める |
| **火災** | 中 | 消火器を使う |
| **デスリチュアル** | 高 | タイマー前にロウソクを取り除く（各ロウソクがSanityを減少させる） |
