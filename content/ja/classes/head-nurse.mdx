# ヘッドナース — B/Sティアクラス

インベントリ拡張、パッシブサニティ回復、アイテム生成を融合した、バランスの取れたプレミアムユーティリティクラス。ティアランキングは情報源によって異なり、IGNはその汎用性の高いキットからSティアに位置づけていますが、TechWiserはRobuxのみの価格とより安価な代替品との競合からBティアにランク付けしています。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 80 Robux（情報源により異なる：80〜190 Robux） |
| **ティア** | B / S（議論あり） |
| **役割** | サポート / ユーティリティ |
| **レアリティ** | プレミアム（Robux） |
| サニティ | 中（レベル1）、高（レベル2〜3） |
| ユーティリティ | 非常に高い |

## コアアビリティ

最大インベントリ容量+3 — デフォルトの上限を超えて3つの追加アイテムを所持でき、補給管理とチームサポートにおいて大きな戦略的優位性をもたらします。

### レベル1
最大インベントリ容量+3

### レベル2
最大インベントリ容量+3、ボーナスサニティ+10

### レベル3
最大インベントリ容量+3、全アイテムを持った状態で各ラウンドを開始

## 戦略のヒント

ヘッドナースの最大の強みは汎用性です。レベル1でのインベントリスロット+3は、すでにナースのレベル3の上限に匹敵しており、そこからさらに強力になります。レベル2では、ボーナスサニティ+10により、回復のために常に後退することなく前線サポートとして機能できます。レベル3はこのクラスが真に輝く場所です — 全ラウンドを完全なアイテムロードアウトで開始できるため、補給品を探す無駄な時間がゼロになります。

Sティアの根拠は、レベル3のアイテム生成とインベントリ容量の組み合わせから来ており、実質的にチーム全体の自給自足型サポートハブとして機能します。Bティアの根拠はよりシンプルです：80以上のRobuxを費やすなら、外科医やドクターのアップグレードに投資した方がより直接的なゲームプレイの影響をもたらすと主張できます。チームの補給を管理するサポートプレイヤーとして主にプレイする場合、ヘッドナースは卓越しています。能動的な治療役を好む場合は、Robuxを他の場所に配分することを検討してください。
