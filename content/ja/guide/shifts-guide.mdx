# Animal Hospital 全シフトガイド — シフトシステムと難易度の解説

シフトシステムは、Animal Hospitalの進行の骨組みです。プレイする各ラウンドは「シフト」であり、動物の患者をスクリーニングし、異常を特定し、タイマーがゼロになるまで生き残ろうとする時間制限付きセッションです。このガイドでは、**シフトの仕組み**、**シフトの総数**、**難易度の上昇方法**、そして**最も厳しいシフトを生き残る方法**を詳しく解説します。

---

## シフトとは？

**シフト**とは、Animal Hospitalの1回の完全なラウンドのことです。決められた時間（通常5〜8分）があり、その間に以下のことが起こります：

1. **患者が到着** — 受付窓口に来る患者全員をスクリーニングする必要があります。
2. **異常が出現** — 患者に変装して現れるため、特定して拒否する必要があります。
3. **モンスターが徘徊** — 病院内を歩き回るため、避けるか対処する必要があります。
4. **正気度が減少** — アイテムやクラスアビリティを使って管理する必要があります。
5. **タイマーがゼロに** — 最後まで生き残ることでシフト完了となります。

シフト中に正気度が0になると、あなたは**排除**され、そのシフトは完了したことにはなりません。

---

## シフトは全部でいくつあるのか？

コミュニティのテストとプレイヤーの報告に基づくと、Animal Hospitalはエンディングが発生するまでに**約15〜20のシフト**があります。正確な数はいくつかの要因によって異なる場合があります：

| 要因 | 総シフト数への影響 |
|--------|----------------------|
| **異常識別率** | 1シフトあたりにより多くの異常を捕まえると、必要な総シフト数が減る可能性があります |
| **ゲームのアップデート** | 開発者がパッチでシフト数を調整する場合があります |
| **難易度修飾子** | 一部のサーバー設定が進行速度に影響を与える可能性があります |

ほとんどのプレイヤーは**シフト15〜18**あたりでエンディングに到達すると報告しています。シフトシステムは徐々に難易度が上がるように設計されているため、最終シフトは最初のシフトよりもはるかに難しくなります。

## シフト全体の難易度スケーリング

Animal Hospitalの難易度は、**3つの主要な軸**に沿ってスケーリングします：

### 1. 異常の出現頻度

初期のシフトでは異常は比較的少なく、患者のウェーブあたり1〜2体程度です。ゲーム中盤（シフト8〜12）では、ウェーブあたり2〜3体の異常が期待できます。終盤のシフト（シフト15以降）では、1つのウェーブで**4体以上の異常**が襲ってくる可能性があります。

### 2. 異常の種類

すべての異常タイプが最初から出現するわけではありません。新しい、より危険な異常タイプは、**シフトが進むにつれてアンロック**されます：

| シフト範囲 | アンロックされる異常タイプ |
|-------------|----------------------|
| **シフト1〜3** | 基本的な外見の異常（三つ目、痙攣、うつろな顔） |
| **シフト4〜7** | 写真の異常（呪いの写真、ノイズ写真、目の違い） |
| **シフト7〜10** | 監視カメラの異常（虚無の身体、カメラを見つめるもの、影の人物） |
| **シフト10〜13** | レア異常および高度な検出タイプ |
| **シフト14以降** | 最もレアで欺瞞に満ちたものを含む、すべての異常タイプ |

この段階的なアンロックは、初期のシフトが学習曲線であることを意味し、後期のシフトは異常の全リストに対する習熟度を試すものとなります。

### 3. モンスターの激しさ

シフトが進むにつれて、モンスターはより攻撃的で頻繁になります：

| シフト範囲 | モンスターの活動 |
|-------------|-------------------|
| **シフト1〜5** | モンスターの遭遇はまれ。主にベッドモンスターとヘッドバンガー |
| **シフト6〜10** | スキンウォーカーがより頻繁に出現。ストーカーが執拗に |
| **シフト11〜15** | 複数のモンスタータイプが同時に活動。ハイダーが一般的に |
| **シフト16以降** | ほぼ常時モンスターの圧力。最終シフトでは特別な遭遇あり |

## シフトの流れ — 各シフトで起こること

すべてのシフトは一貫した構造に従いますが、詳細は時間とともに激化していきます：

### フェーズ1：開始（0〜60秒）

- プレイヤーは病院にスポーンし、持ち場につきます。
- **物資を集める** — 休憩室からコーヒーを、物資部屋からチョコレートを取ります。
- **役割を割り当てる** — 受付、監視カメラ、回復、モンスター防御の担当を決めます。
- 最初の患者が受付窓口に到着し始めます。

### フェーズ2：患者のスクリーニング（60秒〜シフト中盤）

- ゲームプレイの大部分はここで行われます。
- **3ステッププロトコル**を使って各患者をスクリーニングします：外見 → 写真 → 監視カメラ。
- 発見に基づいて**入院または拒否**します。
- 入院した患者を治療室で**治療**します。
- **正気度を監視**し、必要に応じてアイテムを使用します。

### フェーズ3：激化（シフト中盤〜残り2分）

- このフェーズでは**異常の出現頻度が増加**します。
- **モンスターの遭遇がより一般的に**なります。
- **環境イベント**がトリガーされる場合があります（明滅、ドアが閉まる音など）。
- ほとんどのプレイヤーがここで正気度を失います — 集中力を保ち、アイテムを使用してください。

### フェーズ4：ラストスパート（残り2分）

- ゲームはすべてを投げつけてきます。
- **最大の異常プレッシャー**。
- **複数のモンスターが同時に活動**する可能性があります。
- **生き残ることが唯一の目標** — 完璧なスクリーニングは気にせず、ただ生き延びてください。
- 正気度が健康なら乗り切ってください。低い場合は、チョコレートを食べ、すべての脅威を避けてください。

### フェーズ5：シフト完了（タイマーが0に）

- あなたが生きていれば、シフトは**完了**とみなされ、合計にカウントされます。
- ロビーに戻り、獲得したAnimalコインとドルを使うことができます。
- 次のシフトは前回より**わずかに難しく**なります。

## 主要なシフトマイルストーン

特定のシフトは、ゲーム内での重要な変化を示しています。注目すべき**主要なマイルストーン**は以下の通りです：

| シフト | マイルストーン | 変更内容 |
|-------|-----------|--------------|
| **シフト1** | チュートリアルフェーズ | 基本的な異常のみで、モンスターの出現頻度も低い。操作方法とスクリーニングプロトコルを学びましょう。 |
| **シフト3〜4** | 写真の異常がアンロック | 外見だけでなく、デスク上のポラロイド写真を確認する必要があります。 |
| **シフト6〜7** | 監視カメラの異常がアンロック | 待合室のカメラが不可欠になります。専任の監視担当を割り当てましょう。 |
| **シフト9〜10** | ゲーム中盤のプレッシャーテスト | 異常の出現頻度が著しく跳ね上がります。カジュアルなプレイヤーが定期的に死に始める場所です。 |
| **シフト12〜13** | レア異常が出現 | 新しく検出が難しい異常タイプが出現します。スクリーニングはほぼ完璧である必要があります。 |
| **シフト15** | 終盤の閾値 | ほぼ最大難易度。ウェーブごとに複数の異常、絶え間ないモンスターの脅威。 |
| **シフト17〜20** | 最終シフト | 固有のストーリーイベント、クライマックスシーケンス、そしてゲームのエンディング。 |

## シフト難易度の概要

| シフト範囲 | 難易度 | 異常の出現頻度 | モンスターの活動 | 正気度への圧力 |
|-------------|-----------|-------------------|-----------------|-----------------|
| **1〜3** | ⭐ 易しい | 低い（ウェーブあたり1〜2） | まれ | 低い |
| **4〜6** | ⭐⭐ 普通 | 中程度（ウェーブあたり2） | 時々 | 中程度 |
| **7〜9** | ⭐⭐⭐ 難しい | 中〜高（ウェーブあたり2〜3） | 頻繁 | 中〜高 |
| **10〜12** | ⭐⭐⭐⭐ 激しい | 高い（ウェーブあたり3） | 非常に頻繁 | 高い |
| **13〜15** | ⭐⭐⭐⭐ 非常に激しい | 非常に高い（ウェーブあたり3〜4） | 常時 | 非常に高い |
| **16〜20** | ⭐⭐⭐⭐⭐ 極限 | 最大限 | ほぼ常時 | 極限 |

## 終盤シフトのサバイバルのコツ

後半のシフトに到達するには、**機械的なスキル、ゲームの知識、チームの連携**の組み合わせが必要です。最も重要な戦略を以下に紹介します：

### 1. 3つの検出方法をすべて習得する

シフト10以降では、3つのスクリーニング方法すべてに習熟している必要があります：

- **外見** — すべての視覚的異常を暗記します（痙攣、多眼、うつろな顔など）。
- **写真** — 呪いの写真を避けるため、デスク上の写真は必ず拾う前に確認します。
- **監視カメラ** — 待合室カメラ専任の監視担当を割り当てます。

基礎を復習する必要がある場合は、[初心者ガイド](/guide/beginner-guide/)をご覧ください。

### 2. バランスの取れたチームを構築する

終盤のシフトはソロではほぼ不可能です。明確な役割を持つチームを構築しましょう：

| 役割 | 最適なクラス | 主な役割 |
|------|-----------|-------------|
| **受付係** | Secretary（事務員） | 患者のスクリーニング、入院/拒否、正気度の回復 |
| **監視カメラ担当** | 任意 | 待合室カメラを常時監視し、異常をコールする |
| **ヒーラー** | Surgeon（外科医）またはDoctor（医師） | 患者の治療、チームの正気度を維持 |
| **セキュリティ** | Secret Agent（秘密捜査官）またはIntern+Gun（実習生+銃） | スキンウォーカー、ハイダー、その他のモンスターに対処 |
| **柔軟対応** | Psychologist（心理学者） | サポート役、アイテム効果を倍増させてチームの正気度を高める |

### 3. 正気度を積極的に管理する

終盤のシフトでは、正気度の減少はこれまで以上に速くなります。積極的な戦略が必要です：

- **正気度を30%未満に下げない** — この閾値を下回ると、視覚的な歪みにより異常検出が信頼できなくなります。
- **チョコレートを備蓄する** — シフトの初期および中盤のフェーズで見つけたチョコレートはすべて取得しましょう。
- **コーヒーを積極的に使用する** — 正気度が下がるのを待たず、患者の合間にコーヒーを啜ってバッファーを維持します。
- **正気度重視のクラスでプレイする** — Surgeon、Doctor、Psychologistはすべてパッシブで正気度を回復します。

正気度のメカニクスの完全な詳細については、[正気度管理ガイド](/guide/sanity-management-guide/)をご覧ください。

### 4. 損失を切り上げるタイミングを知る

終盤のシフトでは、**精度よりも生存が重要**です。患者に不安がある場合は：

- **シャッターを閉じる** — 念のため閉じるのが無難です。
- **1人の患者に5秒以上費やさない** — 次の異常がすでに窓に来ているかもしれません。
- **すべての異常を捕まえることより、自分自身の正気度を優先**する。

### 5. モンスターのパターンを学ぶ

各モンスターには対策があります。終盤のシフトではこれらの対策を知ることが不可欠になります：

| モンスター | 対策 | 優先度 |
|---------|---------|----------|
| **Skinwalker** | 銃（20発）またはX-Taser | 最高 — プレイヤーを殺します |
| **Bed Monster** | ベッドを飛び越えるかメープルシロップを使用 | 高 — 治療室を塞ぎます |
| **Hiders** | ロビーへ逃げるか銃 | 高 — 予測不可能です |
| **The Stalker** | それを見ない | 中 — 正気度の減少が蓄積します |
| **Don't Look Up** | 目薬 | 中 — 持続的な正気度減少 |
| **Camera Anomaly** | すぐにExitを押す | 中 — 見逃すと-20正気度 |
| **Head Banger** | シャッターを閉じる | 低 — 遅くて予測可能 |
| **Death Ritual** | タイマー前にろうそくを除去 | 高 — 蓄積する正気度減少 |

## FAQ — シフトに関する質問

### 死ぬとシフトの進行状況はリセットされますか？

**いいえ。** 死ぬと現在のシフトから排除されます（カウントされません）が、全体的なシフトの進行状況がリセットされることは**ありません**。完了したすべてのシフトは保持され、中断したところから再開します。

### 以前のシフトをやり直すことはできますか？

Animal Hospitalにはシフト選択メニューがありません。1つのシフトを完了すると、常に次のシフトに進みます。ただし、新しいサーバーに参加したり、パーティーでプレイしたりすると、進行の早いシフトに配置される場合があります。

### すべてのサーバーでシフト数は同じですか？

シフトの進行は、サーバーベースではなく**アカウントベース**のようです。あなたの進行状況はサーバー間で引き継がれます。どのサーバーに参加しても、総シフト数は一貫しているはずです。

### シフトのタイマーは毎回同じですか？

基本的なシフトの長さは一定ですが、特定のイベントや異常により、シフトが長くまたは短く**感じられる**場合があります。実際のタイマーは固定されています。

## まとめ

- Animal Hospitalは、エンディングが発生するまでに**約15〜20のシフト**があります。
- **難易度は3つの軸でスケーリング**します：異常の出現頻度、異常の種類、モンスターの激しさ。
- **主要なマイルストーンで新しい異常タイプがアンロック**されます — シフト4前後で写真の異常、シフト7前後で監視カメラの異常、シフト12前後でレア異常。
- **終盤のシフト（10以降）は、専任の役割を持つ協調的なチームが必要**です。
- **正気度の管理が最も重要なサバイバルスキル**です — 正気度を維持するクラスに投資し、アイテムを備蓄しましょう。

これから始めるプレイヤーには、[初心者ガイド](/guide/beginner-guide/)が最初の数回のシフトの基礎をカバーしています。正気度の管理が障壁になっている場合は、[正気度管理ガイド](/guide/sanity-management-guide/)が正気を保つための深い掘り下げを提供します。
