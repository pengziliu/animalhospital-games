# ドクター — Aティアクラス

サージョンを購入するためのコインを貯めるまでの最高の無料オプション。回復ごとに正気度を回復することで、シフト全体を通して健康な状態を保ち続けられます。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 900 アニマルコイン |
| **ティア** | A |
| **役割** | ヒーラー / 患者治療 |
| **レアリティ** | スタンダード |
| 正気度 | 中〜高 |
| 実用性 | 中 |
| 所持金 | なし |


## コアアビリティ

患者を治療した際、正気度を1回復

## アップグレードレベル

### レベル1
患者の治療時に正気度を1回復、ボーナス正気度+10

### レベル2
患者の治療時に正気度を回復、ボーナス正気度+15

### レベル3
患者の治療時に正気度を回復、ボーナス正気度+20

## 戦略のヒント





最高のコストパフォーマンスを誇るヒーラー。患者の治療を通じて正気度を自己維持できます。サージョンに投資する前のゲーム学習に最適です。
