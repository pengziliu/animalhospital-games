# All Classes in Animal Hospital

Classes are unlockable roles in Animal Hospital that provide passive bonuses during shifts. Every player starts as **Intern** for free. Classes are purchased in the lobby Class Shop using **Animal Coins** (earned in-game) or **Robux** (premium currency).

Each class can be upgraded through **3 levels**, with each level adding stronger effects.

## Best Free Class: Surgeon

**Surgeon** (2,500 Animal Coins) is the strongest earnable class. Every heal grants Sanity recovery AND a speed boost. At Level 3, it doubles Sanity gain — making you nearly invincible during shifts.

## Best Paid Class: Secret Agent

**Secret Agent** (Robux) starts with a Gun, the most powerful anomaly weapon. At Level 3, the gun regenerates 1 use per round, effectively giving unlimited shots.

## Recommended Progression Path

1. **Start** as Intern (free)
2. **Save** for Doctor (900 coins) — first Sanity-recovery class
3. **Upgrade** to Surgeon (2,500 coins) — best self-sustaining build
4. **Optional**: Buy Secret Agent (Robux) for maximum anomaly firepower
