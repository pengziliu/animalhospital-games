# Doctor — A Tier Class

The best free option before saving enough for Surgeon. Recovering Sanity on every heal keeps you in a healthy state throughout the shift.

## Overview

| Attribute | Value |
|-----------|-------|
| **Cost** | 900 Animal Coins |
| **Tier** | A |
| **Role** | Healer / Patient Treatment |
| **Rarity** | Standard |
| Sanity | Medium-High |
| Utility | Medium |
| Starter_Money | None |


## Core Ability

Recover 1 Sanity when healing patients

## Upgrade Levels

### Level 1
Recover 1 Sanity when healing Patients, +10 bonus Sanity

### Level 2
Recover Sanity when healing Patients, +15 bonus Sanity

### Level 3
Recover Sanity when healing Patients, +20 bonus Sanity

## Strategy Tips





The best budget healer. Self-sustaining Sanity through patient treatment. Perfect for learning the game before investing in Surgeon.
