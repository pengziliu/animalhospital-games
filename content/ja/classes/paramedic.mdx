# パラメディック — Cティアクラス

パラメディックは生の回復力と引き換えに機動力を得るクラスで、各ラウンド開始時にラージスピードコーラを所持した状態でスタートし、病院内をより速く移動できます。移動速度は対応時間の短縮に役立ちますが、正気度の減少や死亡を防ぐことはできないため、クラス全体の有効性は限定的です。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 250 アニマルコイン |
| **ティア** | C |
| **ロール** | 機動サポート / 応答 |
| **レアリティ** | スタンダード |
| 正気度 | なし（レベル1〜2）、低（レベル3で回復） |
| 実用性 | 中 |

## コアアビリティ

各ラウンドをラージスピードコーラ（6回使用）と共に開始 — これは限られた時間、移動速度を大幅に向上させる消費アイテムで、患者への素早い対応とマップの横断を可能にします。

## アップグレードレベル

### レベル1
ラージスピードコーラ（6回使用）で開始

### レベル2
ラージスピードコーラ（9回使用）で開始

### レベル3
ラージスピードコーラ（9回使用）で開始、+1スピードコーラが毎ラウンド再生

## 戦略のヒント

アニマルホスピタルにおける移動速度は、患者により早く到達し、緊急時により広い範囲をカバーするのに役立ちます。パラメディックのラージスピードコーラはレベル1で1ラウンドにつき6回の速度ブーストを提供し、部屋間を駆け回り、遠くの保管庫から物資を取りに行き、患者を避難させるといった、機動性の高いサポートロールをプレイする場合には意味があります。レベル3では、毎ラウンドの再生により事実上速度ブーストが尽きることはなく、持続的な高速プレイが可能になります。

しかし、速度だけではゲームの核心的な課題を解決できません。正気度を回復させたり、患者を治療したり、脅威を排除することはできません。正気度が低い状態で患者の部屋に到着した高速のプレイヤーは、依然としてブレークダウンのリスクに直面します。パラメディックは、あなたが物流と迅速な対応に集中する間、治療を担当するチームと組み合わせるのが最適です。ソロクラスとしてはその限界が明らかであり、単独の主力というよりはサポートロールの強化剤として考えるべきです。

## 隣接ティアクラスとの比較

| クラス | コスト | 主なメリット |
|-------|------|-------------|
| **ナース** | 10 コイン | インベントリスロット+1 |
| **パラメディック** | 250 コイン | ラージスピードコーラ（6回使用） |
| **ドクター** | 900 コイン | 治療時に正気度回復 |

パラメディックはナースの25倍のコストがかかりますが、根本的に異なる効用 — 速度対インベントリ — を提供します。このトレードオフに価値があるかどうかはプレイスタイル次第です。ドクターは900コインと3.6倍のコストがかかりますが、自己完結型の正気度回復を提供し、生存率とパフォーマンスにはるかに大きな影響を与えます。ほとんどのプレイヤーにとって、250コインをドクターのアンロックに向けて貯金するのが賢明な選択です。パラメディックは楽しく派手なクラスですが、より高ティアのオプションの純粋な実用性と競い合うには苦労します。
