# Todas as Classes no Animal Hospital

As classes são funções desbloqueáveis no Animal Hospital que fornecem bônus passivos durante os turnos. Todo jogador começa como **Estagiário** gratuitamente. As classes são compradas na Loja de Classes do lobby usando **Animal Coins** (ganhas no jogo) ou **Robux** (moeda premium).

Cada classe pode ser melhorada através de **3 níveis**, com cada nível adicionando efeitos mais fortes.

## Melhor Classe Gratuita: Cirurgião

**Cirurgião** (2.500 Animal Coins) é a classe obtível mais forte. Cada cura concede recuperação de Sanidade E um aumento de velocidade. No Nível 3, dobra o ganho de Sanidade — tornando você quase invencível durante os turnos.

## Melhor Classe Paga: Agente Secreto

**Agente Secreto** (Robux) começa com uma Arma, a arma de anomalia mais poderosa. No Nível 3, a arma regenera 1 uso por rodada, efetivamente dando tiros ilimitados.

## Caminho de Progressão Recomendado

1. **Comece** como Estagiário (grátis)
2. **Junte** para Médico (900 moedas) — primeira classe de recuperação de Sanidade
3. **Evolua** para Cirurgião (2.500 moedas) — melhor build autossustentável
4. **Opcional**: Compre Agente Secreto (Robux) para máxima potência de fogo contra anomalias
