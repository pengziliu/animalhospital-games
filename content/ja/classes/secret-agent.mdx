# シークレットエージェント — S Tierクラス

Robuxを消費する意志があるなら、全体的に最強のクラスです。銃により、強力なアノマリーに対処できます。レベル3では、銃がラウンド間で回復します。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 320 Robux |
| **Tier** | S |
| **役割** | 戦闘 / アノマリー排除 |
| **レアリティ** | プレミアム（Robux限定） |
| 正気度 | 中 |
| 実用性 | 高 |
| 初期所持金 | 銃 |


## コアアビリティ

銃を所持した状態でスタート（危険、複数回使用可能）

## アップグレードレベル

### レベル1
銃を所持した状態でスタート（危険、20回使用可能）

### レベル2
銃を所持した状態でスタート（危険、30回使用可能）、正気度+10ボーナス

### レベル3
銃を所持した状態でスタート（危険、30回使用可能）、毎ラウンド1回使用回数が回復、正気度+20ボーナス

## 戦略のヒント



全体的に最強のクラスです。銃はあらゆるアノマリーの脅威を瞬時に排除します。レベル3では、回復する銃により事実上無敵になります。
