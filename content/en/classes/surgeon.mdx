# Surgeon — S Tier Class

The top class in Animal Hospital for non-Robux players. Passive Sanity gain and speed boost on every heal makes you self-sustaining as you treat more patients.

## Overview

| Attribute | Value |
|-----------|-------|
| **Cost** | 2500 Animal Coins |
| **Tier** | S |
| **Role** | Healer / Primary Patient Treatment |
| **Rarity** | Premium (highest coin cost) |
| Sanity | Medium-High |
| Utility | Medium-Low |
| Starter_Money | None |


## Core Ability

Gain Sanity and a short speed boost after healing a patient

## Upgrade Levels

### Level 1
Get Sanity and Short Speed Boost after healing a Patient

### Level 2
Get Sanity and a longer (8s) speed boost after healing a Patient

### Level 3
Get x2 Sanity and a longer (8s) speed boost after healing a patient, +10 bonus Sanity

## Strategy Tips

The best free-to-play class in the game. Prioritize healing patients to trigger the Sanity + speed boost constantly.
