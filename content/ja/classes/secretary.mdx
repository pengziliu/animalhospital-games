# 秘書 — Bティアクラス

フロント業務に最適なクラスである秘書は、通常は正気度を消費する患者の受付タスクを、自己完結型のワークフローに変えます。120アニマルコインという価格は、利用可能なBティアのアンロックの中で最もコストパフォーマンスに優れた一つです。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 120アニマルコイン |
| **ティア** | B |
| **役割** | フロント / サポート |
| **レアリティ** | スタンダード |
| 正気度 | 中〜高 |
| 利便性 | 高（受付） |

## コアアビリティ

患者の受付時に正気度を1回復 — 受付カウンターでの受付成功ごとに正気度を1ポイント回復し、反復タスクによる通常の消費を相殺します。

## アップグレードレベル

### レベル1
患者の受付時に正気度を1回復

### レベル2
患者の受付時に正気度を1回復、ボーナス正気度+5

### レベル3
患者の受付時に正気度を1回復、ボーナス正気度+10

## 戦略のヒント

秘書は、ゲーム内で大部分の時間をフロントで過ごす予定のある場面で優れた能力を発揮します。次々と患者を受け付けることで通常は正気度が急速に削られてしまいますが、秘書の受付ごとの回復により、レベル3ではこれがゼロサム、あるいはプラスのループに変わります。治療を気にすることなく受付業務に完全に集中できるよう、専任の医師や外科医を持つチームとこのクラスを組み合わせましょう。

レベル2では、ボーナス正気度+5により、受付が本格化する前のシフト開始直後の数分間に余裕が生まれます。レベル3のボーナス正気度+10がまさにスイートスポットです — 受付ごとの回復と組み合わせることで、休憩をとることなく無限にフロント業務を続けることができます。チームに秘書がいない場合、受付に患者が殺到し病院全体の動きが遅くなるため、このクラスはチーム効率に驚くほどの影響を与えます。

## 隣接ティアクラスとの比較

| クラス | コスト | 主なメリット |
|-------|------|-------------|
| **ナース** | 10コイン | インベントリスロット+1 |
| **秘書** | 120コイン | 受付時に正気度回復 |
| **ドクター** | 900コイン | 治療時に正気度回復 |

秘書はわずか120コインでBティアの有用性を提供します — 驚異的なコストパフォーマンスです。ドクターは似たようなアクション時の正気度回復メカニズムを持っていますが、そのコストは約8倍です。ただし、ドクターの治療役は一般的に受付業務よりも汎用性が高いと言えます。ナースと比較しても、秘書はたった110コイン追加するだけで効果が飛躍的に向上します。フロント管理を楽しむのであれば、秘書はゲーム内で最高のコストパフォーマンスを誇るアップグレードです。
