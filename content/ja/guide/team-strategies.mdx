# チーム戦略 — 最高の協力クラス組み合わせとロール

Animal Hospitalは**1サーバーあたり最大30人**のプレイヤーをサポートしており、チームの連携が円滑なシフトと完全な混乱の分かれ目になります。このガイドでは、最適なクラス構成、ロール割り当て、コミュニケーション戦術、ソロ戦略、そして連携の取れていないチームが深夜シフトのイベントに圧倒された時のサバイバル方法を解説します。

---

## チーム構成が重要な理由

10種類のクラスがそれぞれ異なる役割を担っており、バランスの取れたチームはランダムなクラスの集まりよりも圧倒的に優れたパフォーマンスを発揮します。適切な組み合わせは以下のことを意味します：

- **より速い患者の処理能力** — 1シフトあたりにより多くの患者をスクリーニングし治療できる
- **より良い正気度の維持** — ヒーラーがチームを生存させる
- **確実な異常排除** — 武器を持ったプレイヤーがモンスターを素早く処理する
- **完全なCCTVカバレッジ** — 監視の死角がない

## ロール割り当てガイド

成功するすべてのチームには明確なロール割り当てが必要です。以下が**6つのコアロール**と、それぞれに適したクラスです：

### 1. 受付係（異常スクリーナー）

**目標**: 3ステップのプロトコルを使用して、フロントデスクですべての患者をスクリーニングする。

| 最適なクラス | 理由 |
|-------------|-----|
| Secretary | チェックインで正気度回復 — スクリーニングの仕事をすることで報酬を得られる |
| Intern（無料） | 初心者には十分な性能；+10のボーナス正気度がバッファーになる |

**職務**: 3ステップのプロトコル（外観 → 写真 → CCTV）に従い、異常にはシャッターを閉め、正常な患者をチェックインする。

### 2. ヒーラー（メディック）

**目標**: チームと患者の正気度をゼロ以上に保つ。

| 最適なクラス | 理由 |
|-------------|-----|
| Surgeon | 最高のヒーラー — 回復のたびに正気度+スピード。自己完結型。 |
| Doctor | 優秀なヒーラー — 回復時に正気度が回復。Surgeonよりも手頃な価格。 |
| Psychologist | 正気度効果2倍 — コーヒーの一口、チョコレートのひと噛みをすべて2倍にする。 |

**職務**: 患者を回復し、チームメイトを回復し、緊急時に正気度アイテムを分配する。

### 3. モンスターハンター（セキュリティ）

**目標**: チームの正気度を奪ったり患者を殺したりする前に、徘徊する脅威を排除する。

| 最適なクラス | 理由 |
|-------------|-----|
| Secret Agent | 20発の銃 — 最高ダメージ、最長射程。 |
| Security | X-Taser（5回使用） — 非致死性だが、脅威をスタンさせるのに効果的。 |

**職務**: 廊下をパトロールし、スキンウォーカーの呼び出しに対応し、ハイダーを排除し、患者室を警備する。

### 4. 補給係（兵站）

**目標**: 必要な時にチームが常に正気度アイテムとユーティリティツールを持っているようにする。

| 最適なクラス | 理由 |
|-------------|-----|
| Head Nurse | 最大インベントリ+3 = 誰よりも多くのアイテムを持ち運べる。 |
| Nurse | 最大インベントリ+1 — お買い得な代替品。 |

**職務**: チョコレート、コーヒー、目薬、メープルシロップを集め、必要としているプレイヤーに分配する。

### 5. スカウト（偵察）

**目標**: CCTVカメラを監視し、チームに異常や脅威を知らせる。

| 最適なクラス | 理由 |
|-------------|-----|
| Paramedic | スピードコーラにより、カメラバンクと受付エリア間を素早く移動できる。 |
| Intern | お買い得な選択肢 — モニターの前に座っているだけでいい。 |

**職務**: 主に待合室のCCTVを見守り、CCTVの異常をコールし、ストーカーの動きを追跡する。

### 6. フレックス / オールラウンダー

**目標**: 空白を埋め、どこで起きても緊急事態に対応する。

| 最適なクラス | 理由 |
|-------------|-----|
| Psychologist | 正気度効果2倍により、どんな状況でも効果的。 |
| Nurse | 専門的なロールなしのインベントリスロット+1 — 純粋な柔軟性。 |

## コミュニケーションのヒント

ボイスチャットとゲーム内テキストチャットは、最も強力なツールです。これらのコールアウト（掛け声）を使用しましょう：

### 標準的なコールアウト

| 状況 | 言うべきこと |
|-----------|-------------|
| 異常を発見した時 | "窓口に異常 — [外観/写真/CCTV]" |
| モンスターを目撃した時 | "廊下にスキンウォーカー — 銃が必要！" |
| 正気度低下の警告 | "正気度が20 — チョコレートが必要" |
| アイテムが利用可能 | "3号室近くのカウンターにチョコレートあり" |
| カメラがダウン | "待合室のカメラが壊れた — 誰か直して" |
| ヒーラーが必要 | "2号室の患者が死にそう — Surgeonが必要" |

### コミュニケーションのルール

1. **受付のコールは1人に** — 同じ患者について3人が叫ぶのを避ける
2. **コールアウトにクラス名を使用する** — "ヒーラー来て"より"4号室へSurgeon"の方が早い
3. **正気度レベルを自発的にコールする** — 5になるまで待たない
4. **キルを確認する** — "スキンウォーカー排除"でチームが心配するのをやめられる

## ソロ vs チーム戦略

### ソロプレイ

ソロプレイも完全に可能であり、適切なクラスを選べば効率的に進めることができます。

| 戦略 | 詳細 |
|----------|---------|
| **最高のソロクラス** | **Surgeon** — 自己完結型の正気度回復により、チームメイトに依存しない |
| **お買い得ソロクラス** | **Doctor** — 900 AC、回復で正気度回復、ほとんどのシフトには十分 |
| **プレイスタイル** | 自分で患者をスクリーニング（3ステッププロトコル）、患者を回復して正気度を得る、患者の合間に無料のコーヒーを摂取 |
| **主要アイテム** | チョコレート（合計正気度60）、コーヒー（無料）、緊急用の銃（$410） |
| **弱点** | チームなしでのスキンウォーカーとの遭遇は危険です。銃かテーザーを準備しておくこと。 |

### チームプレイ

| 利点 | 詳細 |
|-----------|---------|
| **並行スクリーニング** | 1人のプレイヤーがCCTVを確認し、もう1人が受付でスクリーニングする |
| **分散ヒーリング** | 複数のヒーラーがより多くの患者を同時にカバーする |
| **モンスター対応** | 武器を持ったプレイヤーが脅威を排除している間、他のプレイヤーはスクリーニングを続ける |
| **アイテム共有** | 補給係が正気度回復が切れる人がいないようにする |
| **正気度のセーフティネット** | 1人のプレイヤーが0になっても、他のプレイヤーがシフトを続ける |

## 深夜シフトの混乱への対処

シフトの終わりは、すべてがエスカレートする時です：より多くの異常、複数のモンスターの徘徊、そしてプレイヤーの正気度の低下。サバイバル方法は以下の通りです：

### 準備（シフト開始時）

- ショップで**チョコレート**と**コーヒー**を買いだめする
- 少なくとも1人のプレイヤーが**銃**または**テーザー**を持っていることを確認する
- 開始時から専用の**CCTV監視係**を配置する
- ハイリスクエリアの近くに**消火器**を配置する

### シフト中盤（安定フェーズ）

- 3ステップのスクリーニングプロトコルを維持する — 怠けないこと
- ヒーラーはチームの正気度を自発的に満タンにする
- 補給係は混乱フェーズの前にアイテムを再分配する

### 深夜シフト（混乱フェーズ）

| 脅威 | 対応 |
|--------|----------|
| **同時に複数の異常** | 全員が割り当てられたロールに集中する — スクリーニングを重複しない |
| **スキンウォーカー + ハイダーが同時出現** | Secret Agentがスキンウォーカーを処理；テーザーユーザーがハイダーを処理；他はスクリーニングを続ける |
| **チームの正気度が低下** | Psychologistが2倍のチョコレートを使用；Surgeonが全員を回復；Head Nurseが残りのアイテムを分配 |
| **異常ウェーブ中にカメラが故障** | 外観+写真のみに頼る — 脅威がアクティブな間に修理して時間を無駄にしない |
| **火災発生** | すべてを放下って消火器をつかむ — 火は急速に広がり、患者にダメージを与える |
| **Don't Look Upがアクティブ** | 目薬を持った1人が即座に対処；他は下を向いて作業を続ける |

### 緊急時の優先事項

```
1. 生存 — 正気度を0以上に保つ。アイテムを使用する。
2. 患者を保護 — 死亡を防ぐために瀕死の患者を回復する。
3. 脅威を排除 — 銃/テーザーでアクティブなモンスターを対処。
4. 患者をスクリーニング — 安全になったら通常業務を再開する。
```

## クラスシナジーマトリクス

一部のクラスの組み合わせは、個々の能力を超えた強力なシナジーを生み出します：

| 組み合わせ | シナジー効果 |
|-------|---------------|
| **Psychologist + Head Nurse** | チョコレートの正気度効果2倍/+3インベントリで持ち運べる = 大量の正気度備蓄 |
| **Surgeon + Paramedic** | 回復 → スピードブースト → スピードコーラ = ゲーム内最速のヒーラー |
| **Secretary + Psychologist** | チェックイン正気度獲得 × 2 = Secretaryが受付で正気度を失うことはほぼない |
| **Secret Agent + Security** | 銃 + テーザー = 異なる脅威レベルに対する2つの戦闘オプション |
| **Doctor + Nurse** | お買い得ヒーラー + 追加インベントリ = 安価だが効果的なサポートデュオ |
