# Animal Hospitalの全異常

異常とは、通常の動物患者に紛れ込んで変装したモンスターのことです。外見、写真の異常、CCTV映像、または病院を徘徊するモンスターに遭遇することで識別できます。主な識別方法は3つあります：外見（視覚検査）、写真（ポラロイド確認）、CCTV（監視カメラ確認）です。

## 検出方法

患者に変装した異常を識別するには、**主に3つの方法**があります：

1. **外見** — 受付デスクで患者を直接見る
2. **写真** — ポラロイドを撮り、デスクで現像されるのを待つ
3. **CCTV** — 監視カメラ、特に待合室のカメラを確認する

## 外見の異常

受付デスクで患者を直接見ることで確認できます。カメラや写真を確認する必要はありません。

| 異常 | 難易度 | 対処法 |
|---------|-----------|--------|
| **痙攣 / 四肢のけいれん** | 中 | すぐにシャッターを閉じる |
| **三つの目** | 易 | すぐにシャッターを閉じる |
| **うつろな顔 / うつろな患者** | 易 | すぐにシャッターを閉じる |
| **見開いた目と不気味な笑顔 / 大きな歯** | 易 | すぐにシャッターを閉じる |
| **人間の歯** | 易 | すぐにシャッターを閉じる |
| **不揃いな目と鋭い歯 / 粗く描かれた患者** | 中 | すぐにシャッターを閉じる |

## 写真の異常

直接は見えません。患者のポラロイド写真にのみ現れます。一部の呪われた写真は拾うと正気度（Sanity）を10消費します。

| 異常 | 難易度 | 対処法 |
|---------|-----------|--------|
| **違う目（写真）** | 難 | 写真を拾う前にシャッターを閉じる |
| **違う耳（写真）** | 中 | 写真を拾う前にシャッターを閉じる |
| **不正確な写真** | 易 | 写真を拾う前にシャッターを閉じる |
| **不自然な写真** | 易 | 写真を拾う前にシャッターを閉じる |
| **ノイズ写真** | 易 | 写真を拾う前にシャッターを閉じる |
| **呪われた写真** | 中 | 写真を拾う前にシャッターを閉じる |

## CCTVの異常

直接も写真でも見えません。CCTV監視カメラでのみ確認できます。

| 異常 | 難易度 | 対処法 |
|---------|-----------|--------|
| **検閲された目 / JPEG目** | 易 | シャッターを閉じる |
| **不自然な体 / 四肢の伸長** | 易 | シャッターを閉じる |
| **うつろな顔（CCTV）** | 難 | シャッターを閉じる |
| **カメラを見つめる / カメラ監視者** | 中 | シャッターを閉じる |
| **虚無の体 / 影の患者** | 易 | シャッターを閉じる |
| **スキンウォーカー（CCTV）** | 中 | シャッターを閉じる |
| **不揃いな耳または目（CCTV）** | 激難 | シャッターを閉じる |
| **痙攣（CCTV）** | 難 | シャッターを閉じる |

## モンスターと危険要素

病院の廊下や患者用病室を徘徊する危険要素と脅威。

| モンスター | 危険度 | 対処法 |
|---------|--------|---------------|
| **ベッドモンスター / ベッドの下のモンスター** | 高 | 赤いラインを避け、ベッドを飛び越える。患者を治療するとモンスターは去る。メープルシロップが大好き — 与えれば安全に去っていく。 |
| **ヘッドバンガー / ヘッドノッカー** | 低 | 頭を振り上げて頭突きをしてくる間にシャッターを閉じる。モンスターが消えるまで通常通りシャッターを使用できない。不便だが致命的ではない。 |
| **スキンウォーカー（病院徘徊）** | 極高 | 銃やテーザーでモンスターを撃つ。Eキーを長押しして犠牲者から引き剥がすか、攻撃された場合はEキーを連打してパンチすることもできる。あなたや患者が食べられる前にタイマーを倒す。 |
| **ストーカー** | 中 | 可能であれば目を合わせない。見てしまうと正気度（Sanity）が10減少。大きな問題ではないが、蓄積する。 |
| **テンドリル（触手）** | 中 | タイマーが切れる前に患者の手術を完了する。ミニゲームが完了すると触手は消滅する。間に合わない場合、患者は死亡する。 |
| **カメラ異常 / 悪魔猫のジャンプスケア** | 中 | IGN：画面下部の終了ボタンを時間内に押し、カメラを修理する。Eurogamer：カメラを使用しない以外に回避方法はない。ジャンプスケアは正気度を20消費する。 |
| **上を見るな / 肉の天井** | 中 | 上を見てはいけない！見ると継続的に正気度が減少する。除去方法：目薬を持ち、内側を向いてドアを開け、ドアの上に乗り、「治療を受け入れる」のテキストが表示されるまでゆっくりとモンスターを見る。 |
| **ハイダー / 半透明の男** | 高 | IGN：荒い息遣いに耳を傾け、部屋をスキャンし、目と笑顔を探す。避けるか銃・テーザーを使用する。Eurogamer：ロビーへ逃げると、半透明の男は煙と共に消滅する。 |
| **デスリチュアル（死の儀式）** | 高 | タイマーを倒して部屋に到達し、すべてのろうそくを除去する。各ろうそくの除去は自身の正気度を消費するが、失敗すると患者が死亡する。 |
| **スライム** | 低 | スライムの塊一つ一つにインタラクトして掃除する。 |
| **火災** | 中 | 消火器（ショップへのドアの右側）を手に取り、火を消火する。 |


## 異常スクリーニングチェックリスト

患者を入院させる前に、このチェックリストに従ってください：

- [ ] 患者を直接4〜5秒間見る
- [ ] 写真を撮り、デスクで現像されるのを待つ
- [ ] 待合室のCCTVカメラを確認する
- [ ] 写真と患者の外見を比較する
- [ ] 少しでも違和感があれば、すぐにシャッターを閉じる

## プロのコツ

- **待合室のCCTV**は最も重要なカメラ — すべての患者を確認すること
- **呪われた写真**は拾うと正気度を10消費する — まずシャッターを閉じること
- 一部の異常は**一つの検出方法**にしか現れない — 常に3つすべてを使用すること
- **音の手がかり**（スキンウォーカーの咆哮、ハイダーの息遣い）が危険を知らせてくれる
- カメラを正常に保つ — 故障したカメラはあなたを盲目にする
