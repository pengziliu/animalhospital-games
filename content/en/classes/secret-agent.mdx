# Secret Agent — S Tier Class

The overall best class if willing to spend Robux. The Gun allows you to deal with any powerful Anomalies. At level 3, the gun regenerates between rounds.

## Overview

| Attribute | Value |
|-----------|-------|
| **Cost** | 320 Robux |
| **Tier** | S |
| **Role** | Combat / Anomaly Elimination |
| **Rarity** | Premium (Robux exclusive) |
| Sanity | Medium |
| Utility | High |
| Starter_Money | Gun |


## Core Ability

Start with a gun (dangerous, multiple uses)

## Upgrade Levels

### Level 1
Start with a gun (dangerous, 20 uses)

### Level 2
Start with a gun (dangerous, 30 uses), +10 bonus Sanity

### Level 3
Start with a gun (dangerous, 30 uses) that regenerates 1 use each round, +20 bonus Sanity

## Strategy Tips



The strongest class overall. The Gun eliminates any anomaly threat instantly. At Level 3, the regenerating gun makes you practically unstoppable.
