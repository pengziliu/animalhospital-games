# アニマルホスピタルの全クラス

クラスは、アニマルホスピタルにおいてシフト中にパッシブボーナスを提供する解放可能なロールです。すべてのプレイヤーは無料で**インターン（Intern）**としてスタートします。クラスはロビーのクラショップで**アニマルコイン**（ゲーム内で獲得）または**Robux**（プレミアム通貨）を使用して購入します。

各クラスは**3レベル**までアップグレード可能で、レベルごとにより強力な効果が追加されます。

## 最高の無料クラス：サージェン

**サージェン（Surgeon）**（2,500アニマルコイン）は、獲得可能なクラスの中で最強です。回復するたびに正気度が回復し、さらにスピードブーストを獲得します。レベル3では正気度の獲得量が2倍になり、シフト中はほぼ無敵状態になります。

## 最高の有料クラス：シークレットエージェント

**シークレットエージェント（Secret Agent）**（Robux）は、最も強力なアノマリー武器である銃を初期装備しています。レベル3になると、1ラウンドごとに銃の使用回数が1回再生され、実質的に無制限に発砲できるようになります。

## 推奨進行ルート

1. **スタート** — インターン（無料）で開始
2. **貯金** — ドクター（900コイン）を狙う。初の正気度回復クラス
3. **アップグレード** — サージェン（2,500コイン）へ。最高の自己完結型ビルド
4. **オプション** — シークレットエージェント（Robux）を購入し、最大のアノマリー火力を確保
