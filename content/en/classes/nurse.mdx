# Nurse — C Tier Class

The cheapest unlockable class in the game, offering a modest inventory capacity boost at a very low price. Best viewed as an early stepping stone rather than a long-term pick.

## Overview

| Attribute | Value |
|-----------|-------|
| **Cost** | 10 Animal Coins |
| **Tier** | C |
| **Role** | Support / Inventory |
| **Rarity** | Standard |
| Sanity | None |
| Utility | Low |

## Core Ability

+1 Max Inventory Capacity — carry one additional item in your inventory beyond the default limit.

## Upgrade Levels

### Level 1
+1 Max Inventory Capacity

### Level 2
+2 Max Inventory Capacity

### Level 3
+3 Max Inventory Capacity

## Strategy Tips

A single inventory slot for 10 coins is essentially nothing in terms of practical advantage. The Nurse class is best used as a placeholder or transitional pick while you save up for a more impactful class like Doctor or Secretary. At Level 3, the three extra slots can be mildly useful for hoarding healing supplies, but by that point you could have nearly afforded a Doctor unlock instead.

The Nurse has zero Sanity recovery, meaning you will burn through your sanity bar just as fast as the Intern starter class — but without the Intern's passive Sanity buffer. Only consider running Nurse if you specifically need extra inventory slots for a coordinated team strategy where one player handles supply logistics.

## Comparison to Adjacent Tier Classes

| Class | Cost | Key Benefit |
|-------|------|-------------|
| **Intern** | Free | +10 Bonus Sanity (starter) |
| **Nurse** | 10 Coins | +1 Inventory Slot |
| **Paramedic** | 250 Coins | Speed boost + Inventory |

Compared to the Intern you start with, Nurse trades away a valuable +10 Sanity passive for a single inventory slot. The Paramedic, one tier up in cost, provides movement speed alongside inventory benefits — a far more versatile package. Nurse sits in an awkward middle ground where neither the budget nor the power justify dedicated use.
