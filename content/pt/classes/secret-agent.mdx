# Secret Agent — Classe Tier S

A melhor classe no geral se você estiver disposto a gastar Robux. A Arma permite lidar com qualquer Anomalia poderosa. No nível 3, a arma se regenera entre as rodadas.

## Visão Geral

| Atributo | Valor |
|-----------|-------|
| **Custo** | 320 Robux |
| **Tier** | S |
| **Função** | Combate / Eliminação de Anomalias |
| **Raridade** | Premium (exclusivo Robux) |
| Sanidade | Média |
| Utilidade | Alta |
| Dinheiro_Inicial | Arma |


## Habilidade Principal

Comece com uma arma (perigosa, múltiplos usos)

## Níveis de Melhoria

### Nível 1
Comece com uma arma (perigosa, 20 usos)

### Nível 2
Comece com uma arma (perigosa, 30 usos), +10 de Sanidade bônus

### Nível 3
Comece com uma arma (perigosa, 30 usos) que regenera 1 uso a cada rodada, +20 de Sanidade bônus

## Dicas de Estratégia



A classe mais forte no geral. A Arma elimina qualquer ameaça de anomalia instantaneamente. No Nível 3, a arma que se regenera torna você praticamente imparável.
