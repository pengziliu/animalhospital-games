# ナース — Cティアクラス

ゲーム内で最も安価にアンロックできるクラスで、非常に低い価格で控えめなインベントリ容量のブーストを提供します。長期的な選択肢というより、序盤の踏み台として捉えるのが最適です。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 10 アニマルコイン |
| **ティア** | C |
| **ロール** | サポート / インベントリ |
| **レアリティ** | スタンダード |
| 正気度 | なし |
| 実用性 | 低 |

## コアアビリティ

最大インベントリ容量+1 — デフォルトの制限を超えてインベントリにアイテムを1つ追加で所持できます。

## アップグレードレベル

### レベル 1
最大インベントリ容量+1

### レベル 2
最大インベントリ容量+2

### レベル 3
最大インベントリ容量+3

## 戦略のヒント

10コインでインベントリスロット1つは、実質的なメリットとしてほぼ無意味です。ナースクラスは、ドクターやセクレタリーなどのより影響力のあるクラスのために貯金している間の、プレースホルダーや移行期の選択として最適です。レベル3になると、追加の3スロットは回復アイテムを備蓄するのにそこそこ役立ちますが、その頃にはドクターのアンロックに必要な額にほぼ手が届いているでしょう。

ナースは正気度の回復がゼロであり、インターンのスタータークラスと同じくらいの速さで正気度バーを消費することを意味します — ただし、インターンのパッシブな正気度バッファはありません。1人のプレイヤーが補給の管理を担当する協力プレイの戦略で、特に追加のインベントリスロットが必要な場合のみナースの運用を検討してください。

## 隣接ティアクラスとの比較

| クラス | コスト | 主なメリット |
|-------|------|-------------|
| **インターン** | 無料 | 正気度+10ボーナス（スターター） |
| **ナース** | 10コイン | インベントリスロット+1 |
| **パラメディック** | 250コイン | 移動速度アップ+ インベントリ |

最初から所持しているインターンと比較すると、ナースは貴重な正気度+10のパッシブと引き換えにインベントリスロット1つを得ることになります。コストが1つ上のティアであるパラメディックは、インベントリのメリットに加えて移動速度も提供し、はるかに汎用性の高いパッケージとなっています。ナースは、予算的にもパワーの面でも専用での使用を正当化できない、微妙な中間的な位置にあります。
