# インターン — Dティアクラス

全プレイヤーが最初に始めるデフォルトの開始クラス。ステータスはゲーム内で最も低いですが、パッシブのサニティボーナスは、操作に慣れ、より良いクラスのためにコインを貯めている間の最初の数回のシフトで小さな助けになります。

## 概要

| 属性 | 値 |
|-----------|-------|
| **コスト** | 無料（デフォルトスターター） |
| **ティア** | D |
| **ロール** | 汎用 / 学習 |
| **レアリティ** | デフォルト |
| サニティ | 低〜中 |
| 実用性 | なし |

## コアアビリティ

+10ボーナスサニティ — 基本値を超えて10ポイントの追加サニティを持って各シフトを開始し、クリティカルなサニティレベルに達するまでの時間を少し長くします。

## アップグレードレベル

### レベル1
+10ボーナスサニティ

### レベル2
+15ボーナスサニティ

### レベル3
+20ボーナスサニティ

## 戦略のヒント

スタータークラスとして、インターンは最初の数回のプレイでサニティ枯渇に陥ることなく進められるように設計されています。+10のパッシブ効果により、いくつかのミスが許されたり、休憩前にあと1人の患者を診ることができます。病院のレイアウト、受付の流れ、どのアップグレードのために貯金するかを学ぶことを優先しましょう。

新しいクラスにコインを使わずにアニマルコインを稼ぎ続けるつもりがある場合のみ、インターンをレベル3にアップグレードしてください。レベル3の+20サニティバッファーはまともですが、900コインのドクターの回復時サニティ回復には及びません。ドクターの回復は一時的なバッファーではなく自己完結型です。少なくともナース（10コイン）、理想的にはドクターを買えるようになるまでインターンを使用し、その後すぐに切り替えましょう。

## 隣接ティアクラスとの比較

| クラス | コスト | 主なメリット |
|-------|------|-------------|
| **インターン** | 無料 | +10ボーナスサニティ（スターター） |
| **ナース** | 10コイン | インベントリスロット+1 |
| **ドクター** | 900コイン | 回復時にサニティ回復 |

インターンの唯一の本当の利点は無料であることです。わずか10コインのナースでさえインベントリアップグレードを提供し、900コインのドクターは自己完結型のサニティ回復を導入し、インターンの固定ボーナスを取るに足らないものにします。インターンをチュートリアルクラスとして扱いましょう — 基本を学んでから卒業してください。
