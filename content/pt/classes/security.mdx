# Security — Classe Tier B

A classe Security te dá um X-Taser logo de cara para neutralizar ameaças e eventos disruptivos. No entanto, o taser também está disponível gratuitamente no Reception Office, o que torna o preço de 1.250 Animal Coins difícil de justificar para a maioria dos jogadores.

## Visão Geral

| Atributo | Valor |
|-----------|-------|
| **Custo** | 1.250 Animal Coins |
| **Tier** | B |
| **Função** | Defesa / Controle de Ameaças |
| **Raridade** | Standard |
| Sanidade | Baixa (Nível 1), Média (Nível 2–3) |
| Utilidade | Média-Alta |

## Habilidade Principal

Comece cada rodada com um X-Taser (5 usos) — uma arma de choque que pode incapacitar ameaças e entidades disruptivas no hospital.

## Níveis de Upgrade

### Nível 1
Comece com X-Taser (5 usos)

### Nível 2
Comece com X-Taser (8 usos), +10 de Sanidade Bônus

### Nível 3
Comece com X-Taser (8 usos), +15 de Sanidade Bônus

## Dicas de Estratégia

O maior argumento contra o Security é que o X-Taser aparece gratuitamente no Reception Office toda rodada. Qualquer classe pode pegá-lo, o que significa que você está pagando 1.250 coins pela conveniência de tê-lo em mãos desde o início. Se sua equipe for coordenada, um jogador pode simplesmente pegar o taser gratuito a cada rodada sem dedicar um slot de classe ao Security.

Dito isso, o Security se torna mais atraente em partidas caóticas onde o taser gratuito é pego por outro jogador ou destruído. O upgrade de Nível 2 adicionando 3 usos extras e +10 de Sanidade Bônus torna a classe significativamente mais autossuficiente, e os +15 de Sanidade do Nível 3 significam que você pode patrulhar por mais tempo. Se você joga frequentemente em lobbies cheios com jogadores descoordenados, o acesso garantido ao taser é uma verdadeira melhoria de qualidade de vida.

## Comparação com Classes de Tier Adjacente

| Classe | Custo | Principal Benefício |
|-------|------|-------------|
| **Doctor** | 900 Coins | Sanidade ao Curar |
| **Security** | 1.250 Coins | X-Taser no Spawn |
| **Surgeon** | ~2.000+ Coins | Cura Avançada |

Com 1.250 coins, o Security custa mais que o Doctor, mas oferece um valor menos consistente. A recuperação de Sanidade do Doctor é passiva e está sempre ativa; o taser do Security é situacional e redundante se alguém pegar o gratuito. O Surgeon, embora mais caro, oferece uma utilidade de cura dramaticamente superior. O Security é uma escolha de nicho — forte para jogadores que gostam de patrulhar e lidar com ameaças, mas não recomendado como uma classe de uso geral.
